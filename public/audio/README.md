# Audio files (`public/audio`)

## Rhyme background music

Place **MP3** files in `public/audio/rhymes/` and set each rhyme’s `musicFile` in `src/data/rhymes.js`.

Bundled starter clips:

- `music-soft-1.mp3` … `music-soft-4.mp3`

You can replace them with your own royalty-free instrumentals. Keep filenames in sync with `rhymes.js`.

If both local and the remote fallback fail, the app shows a friendly error on the rhyme card.
